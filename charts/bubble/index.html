Red bubbles indicate the top industries for the selected state. Interactive bubble chart with v2 model integration and actual job quantity calculations by Saalim.

Chart uses USEEIOv2.0/indicators.json and indicators_sectors.csv which is generated by USEEIO_indicators.ipynb using the API https://api.edap-cluster.com/useeio/api   Values represent the impact of US production in 2007.

Note that v2.0 has a different sector schema. So if you've manually got groups of sectors used in displays, it could break them. The changes are to <10% of the sectors, so it may not be apparent immediately. The sectorcrosswalk.csv resides in the useeio-json repo.
